# Project instructions — global-events

## Commit and PR conventions

This repository **overrides** the global "never add AI attribution" rule. The
override applies **only to this repository**.

In this repo, commits and pull requests **should** carry AI attribution:

- End commit messages with the trailer:
  `Co-Authored-By: Claude Opus 4.8 <noreply@anthropic.com>`
- Include the footer in commit messages and PR bodies:
  `🤖 Generated with [Claude Code](https://claude.com/claude-code)`

Everywhere else, the global no-attribution convention still applies.
